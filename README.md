# Tabel Anggota (HTML & CSS Study Case)

Proyek ini adalah implementasi praktis dari materi **Web Design Day 2** yang fokus pada pembuatan dan pengelolaan tabel menggunakan HTML serta styling dasar dengan CSS.

## 📋 Deskripsi Proyek
Repositori ini berisi simulasi halaman daftar anggota (*Members List*) yang mencakup fitur visual seperti:
- Header tabel dengan ikon.
- Kontrol tabel (jumlah baris per halaman dan baris pencarian).
- Tabel data dengan status berwarna (Banned, Pending, Active).
- Tombol aksi (View, Edit, Delete).
- Navigasi paginasi.

## 📚 Materi yang Diimplementasikan
Berdasarkan modul "Web Design Day 2", proyek ini menerapkan konsep-konsep berikut:
- **Elemen Utama Tabel:** Penggunaan tag `<table>`, `<tr>`, `<th>`, dan `<td>`.
- **Pengaturan Jarak:** Implementasi `cellpadding` dan `cellspacing` yang diatur ke nilai `0` untuk tampilan border yang lebih bersih dan modern.
- **Titel & Heading:** Penggunaan header kolom agar teks otomatis menjadi tebal (*bold*) dan berada di tengah secara visual.
- **Styling Background:** Memberikan warna latar belakang yang berbeda pada elemen tertentu untuk meningkatkan keterbacaan data.

## 🛠️ Detail Teknis
- **Struktur:** HTML5.
- **Styling:** CSS Internal (diletakkan di dalam tag `<style>`).
- **Layouting:** Menggunakan Flexbox untuk bagian header dan footer tabel agar responsif terhadap lebar kontainer.

## 🚀 Cara Menjalankan
1. Clone repositori ini atau salin kode dari `index.html`.
2. Buka file `index.html` menggunakan browser pilihan Anda (Chrome, Edge, Firefox, dll).
3. Anda akan melihat tampilan tabel anggota sesuai dengan desain yang telah ditentukan.
